# 상품 데이터베이스 코드 기준

## 기본 구조

내부관리 상품코드는 다음 구조를 사용한다.

```text
대분류-세부분류-속성1-속성2-속성3-순번
```

예시:

```text
TIL-POR-MAT-600600-GRY-001
```

## 운영 기준

고객용 상품명은 사람이 읽기 쉬운 형태를 사용하고, 내부관리 상품코드는 위 코드 체계를 사용한다.

| 구분 | 예시 |
| --- | --- |
| 고객용 상품명 | 600x600 자기질 무광 포세린 타일 그레이 |
| 내부관리 상품코드 | TIL-POR-MAT-600600-GRY-001 |
